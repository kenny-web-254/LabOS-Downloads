LabOS Downloads

One server app. One client app. Fast rollout.

Install LabOS Server on the main hospital lab computer. Install LabOS Client on the other staff computers. The client app connects to the server address you enter on first launch.

Main Lab Computer

LabOS Server

Use this on the one computer that hosts the inventory database and serves requests for the whole lab.

  • Install on the main always-on server PC.
  • After launch, use the app menu to view the network address.
  • Keep this app running while the lab is using the system.
Download Server Installer
Other Lab Computers

LabOS Client

Use this on staff computers that need a desktop app instead of opening LabOS in a browser.

  • Install on request desks, workstations, and admin PCs.
  • On first launch, enter the LabOS Server address once.
  • Example server URL: http://10.96.47.199:3000
